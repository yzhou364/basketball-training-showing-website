# 项目计划 Log

> 篮球教练个人培训课程宣传网站
> 创建日期：2026-04-19

## 1. 项目目标

为篮球教练打造一个对外宣传的个人网站，核心目的：
- 展示教练的专业背景与执教成果
- 介绍其提供的个人培训课程（内容、价格、适合人群）
- 提供便捷的预约/联系方式，促成转化

## 2. 目标用户

- 想请私教的青少年家长（主要决策方）
- 有意提升篮球水平的学员本人（青少年 / 成人）
- 通过搜索或朋友推荐访问网站的潜在客户

## 3. 技术方案（待确认）

**初步推荐**：静态站点 + Tailwind CSS

| 方案 | 优点 | 缺点 |
|------|------|------|
| 纯静态 HTML + Tailwind | 轻量、易部署、零成本、加载快 | 无 CMS，改文案需改代码 |
| Astro | 组件化、SEO 好、仍是静态 | 需要 Node 构建环境 |
| Next.js | 功能完整，适合未来扩展 | 复杂度高，对小站偏重 |

**部署**：GitHub Pages / Vercel（均免费，支持自定义域名）

## 4. 页面结构（初稿）

单页长滚动（SPA 风格）+ 未来可拆分子页：

1. **Hero 首屏** — 教练形象照 + 一句话价值主张 + 预约 CTA
2. **关于教练** — 经历、资质、荣誉、执教理念
3. **培训课程** — 一对一 / 小组课 / 训练营，含价格与适合人群
4. **训练成果** — 照片、视频、学员数据/故事
5. **学员评价** — 家长 & 学员口碑
6. **常见问题** — 费用、频次、场地、退费等
7. **联系与预约** — 微信二维码 / 电话 / 表单

## 5. 已确认决策（2026-04-19）

- **内容**：全部使用占位符（文字 `[Placeholder]`、图片用灰底 SVG），用户后续替换
- **部署**：GitHub Pages（简单、免费）
- **视觉风格**：简约高端（minimal & premium）
  - 配色：off-white 背景 `#fafaf7` + near-black 文字 `#0a0a0a` + 深橙点缀 `#c2410c`（呼应篮球）
  - 字体：Fraunces（serif 标题）+ Inter（sans 正文），经 Google Fonts 引入
  - 大量留白，细边框，无花哨装饰
- **语言**：英文
- **技术栈**：单 `index.html` + Tailwind CSS（Play CDN，零构建）+ 原生 JS
  - 优点：推到 GitHub 即可部署，无 Node/构建依赖
  - 未来若要加博客 / 多页 / 性能优化，可升级到 Astro

## 7. 风险与开放问题

- 素材质量决定最终效果；若无专业拍摄，需引导教练补拍或用合规免费图占位

- 未来是否需要排课系统、在线支付，将影响技术选型
